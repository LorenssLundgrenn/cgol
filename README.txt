A quick project for simulating Conway's game of life.

Program Accepts three optional arguments:
<arg1> - grid width, default - 60
<arg2> - grid height, default - 60
<arg3> - life probability, default - 25

Nifty controls:
esc - pause game
tab - generate random ecosystem
r - wipe grid
left click - create life
right click - erase life
